# SR Launcher V2 — Slavonska Ravnica

Multiplayer launcher za **Farming Simulator 25** koji automatski sinkronizira tvoje modove s modovima na serveru.

## 📥 Instalacija

1. Skini **`SlavonskaRavnica-Setup-1.0.0.exe`** (klikni na njega gore, pa **Download**)
2. Pokreni installer
   > Windows može pokazati "Windows protected your PC" (jer app nije digitalno potpisan) → klikni **More info → Run anyway**
3. Otvori **SR Launcher** s desktopa

## ▶️ Korištenje

1. Klikni **Prijava putem Discorda** → prijavi se i **Authorize**
2. Vrati se u launcher (pusti te unutra ako imaš potrebnu Discord ulogu)
3. Idi na **Modovi → Sinkroniziraj Sve** (ili **Skini Što Fali**) — launcher skine sve potrebne modove
4. Na **Dashboard** klikni **Uđi na Server** i igraj 🚜

## ❓ Nemam pristup?

Ako te launcher ne pusti unutra, nemaš potrebnu Discord ulogu. Obrati se administratorima Discord zajednice.

*Modovi se skidaju automatski, ne trebaš FTP ni ručno postavljati ništa.*
